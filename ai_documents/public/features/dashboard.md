# Dashboard Page

The Dashboard is the main landing page after login, providing an overview of the user's content and quick access to key features.

## Page Location
- URL: `/dashboard`

## Page Sections

### 1. Stats Cards (Top Section)
Three clickable cards showing content counts:
- **Videos Card**: Shows total number of uploaded videos. Click to go to Videos page.
- **Manuals Card**: Shows total number of generated manuals. Click to go to Manuals page.
- **Projects Card**: Shows total number of projects. Click to go to Projects page.

### 2. Quick Actions Card
Two main actions users can take:
- **Process a Video**: Links to the Videos page where users can upload and process videos
- **Create a Project**: Links to the Projects page where users can organize manuals

### 3. Your Journey Card
Visual representation of the documentation workflow:
1. **Upload**: Add video content to vDocs
2. **Generate**: AI processes the video and creates step-by-step documentation
3. **Export**: Download in PDF, Word, HTML and other formats

### 4. Meet Your Guide Card
Introduction to the AI guide assistant:
- Features a prominent bot icon with online indicator
- Explains the guide can help navigate the app, learn features, and get instant help
- "Say Hello" button opens the guide chat panel

## Highlightable Elements
- `nav-dashboard` - Dashboard navigation link
- `nav-videos` - Videos navigation link
- `nav-manuals` - Manuals navigation link
- `nav-projects` - Projects navigation link

## Common User Questions

### "What can I do here?"
The Dashboard shows your content overview (videos, manuals, projects) and provides quick actions to process videos or create projects.

### "How do I get started?"
Follow the journey shown on the dashboard:
1. First, upload a video from the Videos page
2. Process it to generate documentation
3. Export your finished manual in your preferred format

### "What is the guide assistant?"
The guide assistant (me!) is an AI helper that can answer questions, show you where features are located, and help you navigate vDocs. Click "Say Hello" to start chatting.
