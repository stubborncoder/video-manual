# Manual Editor

The manual editor lets you refine and perfect your generated documentation with AI assistance.

## Accessing Manuals

### View Button (Preview)
Click **View** on a manual card to open a read-only preview:
- See the manual as it will appear when exported
- Review content without risk of changes
- From preview, you can click **Evaluate** to assess quality

### Edit Button (Editor)
Click **Edit** on a manual card to open the full editor:
- Modify content with AI assistance
- Manage screenshots
- Export the manual
- Save changes (creates versions)

## Editor Features

### AI Editor Agent
The manual editor includes an AI assistant that helps you:
- Modify content through natural conversation
- Insert new screenshots on request
- Understand context when you select text or images

### Text Interaction
- **Select text** to contextualize the agent - the AI will understand what you're referring to
- Edit step titles and descriptions directly
- Ask the agent to rewrite, expand, or clarify selected content

### Image Interaction
- **Select an image** to contextualize the agent with that image
- **Hover over image + left click** opens contextual menu:
  - **Open full size** - View image at full resolution
  - **Replace from video keyframe** - Swap with a different frame from the source video
  - **Annotate image** - Add annotations, arrows, highlights, or crop the image
  - **Edit caption** - Modify the image's caption text
  - **Ask AI** - Contextualize the manual editor agent with this image
  - **Delete image** - Remove the image from the manual

### Screenshot Management
- View all captured screenshots
- Replace screenshots with better frames from the video
- Request the agent to insert new screenshots
- Annotate images with highlights and crops
- Reorder or delete images as needed

## Saving and Versions

**IMPORTANT: Every save creates a new version.**

- Click **Save** to save your changes
- Each save creates a new version in the history
- View **Version History** to see all saved versions
- You can restore previous versions if needed
- Unsaved changes indicator shows pending edits

### Version History
- Access via the Versions button in the editor
- See timestamp and changes for each version
- Compare versions to see what changed
- Restore any previous version

## Tips for Editing

### Working with the Agent
- Select text before asking the agent to modify it
- Select an image to give the agent visual context
- Be specific in your requests

### Improving Content
- Be specific about what to click
- Mention exact button/menu names
- Include keyboard shortcuts if applicable

### Screenshot Quality
- Replace blurry screenshots using "Replace from video keyframe"
- Use annotations to highlight important areas
- Crop images to focus on relevant content

### Version Management
- Save frequently to create restore points
- Review version history before major changes
- Use descriptive changes so you can identify versions later
