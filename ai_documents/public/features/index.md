# vDocs Features

This section documents all major features of vDocs.

## Core Features

- [Video Upload](video-upload.md) - Upload and manage source videos
- [Manual Generation](manual-generation.md) - AI-powered documentation creation
- [Manual Editor](manual-editor.md) - Edit and refine generated manuals
- [Add Language](add-language.md) - Generate manuals in additional languages
- [Manual Evaluation](evaluation.md) - AI quality assessment and scoring
- [Projects](projects.md) - Organize manuals into collections
- [Export](export.md) - Download documentation in various formats
- [Templates](templates.md) - Customize export styling
- [Bug Tracker](bug-tracker.md) - Report bugs and request features
- [Trash](trash.md) - Recover or permanently delete items
