# vDocs Features

This section documents all major features of vDocs.

## Core Features

- [Video Upload](video-upload.md) - Upload and manage source videos
- [Doc Generation](doc-generation.md) - AI-powered documentation creation
- [Doc Editor](doc-editor.md) - Edit and refine generated docs
- [Add Language](add-language.md) - Generate docs in additional languages
- [Doc Evaluation](evaluation.md) - AI quality assessment and scoring
- [Projects](projects.md) - Organize docs into collections
- [Export](export.md) - Download docs (PDF, Word, Markdown, HTML) or share via public links
- [Templates](templates.md) - Customize export styling
- [Bug Tracker](bug-tracker.md) - Report bugs and request features
- [Trash](trash.md) - Recover or permanently delete items
