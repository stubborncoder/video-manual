# vDocs Documentation

vDocs is an AI-powered application that creates step-by-step documentation from videos. Upload a video and vDocs will automatically generate visual documentation with screenshots and written instructions.

## Quick Start

1. **Upload a video** - Go to Videos page and upload your screen recording
2. **Wait for processing** - AI analyzes the video and extracts key steps
3. **Edit the manual** - Review and refine the generated documentation
4. **Export** - Download in your preferred format (PDF, Markdown, etc.)

## Key Features

- **Video Analysis** - AI extracts key moments and actions from videos
- **Automatic Screenshots** - Captures relevant frames as documentation images
- **Manual Editor** - Rich editor for refining generated content
- **Quality Evaluation** - AI-powered assessment with scores and recommendations
- **Projects** - Organize multiple manuals into collections
- **Export Options** - PDF, Markdown, HTML export formats
- **Templates** - Customize export styling and format
