# vDocs Documentation

**Current Version: v0.2.0-alpha.1** | [What's New](/guides/changelog/v0.2.0-alpha.1.md)

vDocs is an AI-powered application that creates professional documentation from videos. Upload a video and vDocs will automatically generate the right type of document based on your selected format.

## Document Formats

vDocs supports multiple document formats for different use cases:

**Instructional Formats:**
- **Step-by-Step Doc** - Tutorials, how-to guides, training materials
- **Quick Guide** - Condensed overviews for quick reference
- **Reference Document** - Technical documentation with definitions
- **Executive Summary** - High-level briefs for decision-makers

**Report Formats:**
- **Incident Report** - Document issues, damage, or problems with evidence
- **Inspection Report** - Condition assessments and compliance checks
- **Progress Report** - Project status and milestones

## Quick Start

1. **Upload a video** - Go to Videos page and upload your recording
2. **Choose format** - Select the document type that matches your content
3. **Generate** - AI analyzes the video and creates the document
4. **Edit & Export** - Refine the content and download in your preferred format

## Key Features

- **Multi-Format Generation** - Choose from 7 document types (instructional and report formats)
- **Video Analysis** - AI extracts key moments based on selected format
- **Automatic Screenshots** - Captures relevant frames as visual evidence
- **Doc Editor** - Rich editor for refining generated content
- **Quality Evaluation** - AI-powered assessment with scores and recommendations
- **Projects** - Organize multiple docs into collections with sections and chapters
- **Project Compilation** - Combine project docs into a unified document
- **Shareable Links** - Create public links to share docs or compiled projects
- **Export Options** - PDF, Word, HTML, Markdown export formats (including project markdown ZIP)
- **Templates** - Customize export styling and format
- **Bug Tracker** - Report issues and request features *(Coming in Alpha/Beta)*
