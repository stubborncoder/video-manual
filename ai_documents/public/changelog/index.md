## How Versions Work

vDocs uses semantic versioning:
- **Major** (0.x.x) - Pre-release development phase
- **Minor** (x.1.x) - New features and capabilities
- **Patch** (x.x.0) - Bug fixes and improvements
- **Pre-release** (alpha/beta) - Early access versions
