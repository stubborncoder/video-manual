# v0.3.0-alpha.1 - Enhanced Guide Agent

**Release Date:** December 2024

## Overview

This release introduces enhanced UI control capabilities for the Guide Agent, allowing it to programmatically interact with the interface to provide better guided assistance.

## New Features

### Guide Agent UI Control
The Guide Agent can now:
- **Navigate** users to specific pages
- **Highlight** UI elements with pulsing borders
- **Open dropdown menus** programmatically
- **Click buttons** to trigger actions
- **Show informational modals** with tips and explanations
- **Start guided workflows** with multi-step tours

### Menu Item Highlighting
The Guide Agent can now highlight specific options inside dropdown menus:
- Export option
- Share option
- Add Language option
- Evaluate Quality option
- Assign to Project option
- Manage Tags option
- Delete option

### Controllable Components
New component system for guide-controlled UI:
- `ControllableDropdownMenu` - Dropdowns that can be opened programmatically
- `ControllableButton` - Buttons that can be clicked programmatically
- Central registry for managing controllable elements

## Technical Improvements

- Added retry mechanism for click actions after page navigation
- Fixed element ID naming consistency between backend and frontend
- Improved timing handling for component registration

## How to Use

Ask the Guide Agent questions like:
- "How do I add a language to my document?"
- "Show me the export options"
- "Where can I manage tags?"

The agent will navigate you to the right page, open the relevant menu, and highlight the specific option.
