# v0.1.0-alpha.1 - Initial Release

**Released:** November 2024

The first alpha release of vDocs - AI-powered **manual generation** from video.

---

## Core Feature: Video to Manual

Upload a video and vDocs automatically generates a step-by-step manual with screenshots and instructions.

### How It Works

1. **Upload** - Record or upload a video of your process
2. **Process** - AI analyzes the video and extracts key steps
3. **Generate** - Automatic manual creation with screenshots
4. **Edit** - Refine the content in the doc editor
5. **Export** - Download as PDF, Word, HTML, or Markdown

---

## Features

### Document Generation
- Step-by-step manual creation from video
- Automatic screenshot capture at key moments
- AI-written instructions for each step

### Doc Editor
- Rich text editing
- Reorder steps
- Edit screenshots and descriptions
- Add or remove content

### Multi-language
- Generate manuals in multiple languages
- Add translations after initial generation

### Quality Evaluation
- AI-powered assessment of generated manuals
- Scores and recommendations for improvement

### Export Options
- **PDF** - Professional output with styling
- **Word** - Editable .docx format
- **HTML** - Web-ready format
- **Markdown** - Plain text with formatting

### Templates
- Customize export styling
- Pre-configured format presets

---

## Limitations

- **Manuals only** - Single document type (step-by-step manuals)
- **No reports** - Incident, inspection, and progress reports not yet available
- **No projects** - Documents managed individually, no collections

*These limitations are addressed in v0.2.0*
