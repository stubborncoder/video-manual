# Workflow: Create Documentation

Complete guide to creating documentation from a video.

## Overview

1. Upload video
2. Process video
3. Edit doc
4. Export result

## Step 1: Upload Video

**Navigate to Videos page**
- Click **Videos** in sidebar (`nav-videos`)
- Or go to `/dashboard/videos`

**Upload your video**
- Click **Upload Video** button (`upload-video-btn`)
- Select video file from your computer
- Wait for upload to complete
- Video appears in the list with "Uploaded" status

## Step 2: Process Video

**Start processing**
- Find your uploaded video in the list
- Click **Process** button on the video card
- Status changes to "Processing"

**Wait for completion**
- AI analyzes video content
- Screenshots are extracted
- Instructions are generated
- Status changes to "Completed" when done

## Step 3: Edit Doc

**Open the editor**
- Navigate to **Docs** page (`nav-docs`)
- Find your newly created doc
- Click **Edit** button (`first-doc-edit-btn`)

**Review and refine**
- Check generated screenshots
- Edit step descriptions
- Add or remove steps as needed
- Save your changes

## Step 4: Export

**Choose format**
- In editor, click **Export**
- Select format (PDF, Markdown, HTML)

**Configure options**
- Choose template (optional)
- Set quality settings
- Configure page options

**Download**
- Click **Download**
- File saves to your computer

## Checklist

- [ ] Video uploaded successfully
- [ ] Processing completed without errors
- [ ] Doc content reviewed and edited
- [ ] Final export downloaded

## Troubleshooting

**Video won't upload**
- Check file format (MP4, WebM recommended)
- Check file size (under 500MB)
- Try a different browser

**Processing fails**
- Video may be corrupted
- Try re-uploading
- Check video plays correctly locally

**Poor quality output**
- Record higher resolution video
- Slow down actions in recording
- Edit and enhance manually
