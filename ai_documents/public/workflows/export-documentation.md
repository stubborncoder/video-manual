# Workflow: Export Documentation

Guide to exporting your manuals and projects.

## Overview

1. Choose what to export
2. Select format
3. Configure options
4. Download

## Exporting a Single Manual

**Open the manual**
- Go to **Manuals** page (`nav-manuals`)
- Click **Edit** on the manual

**Export**
- Click **Export** button in the editor toolbar
- Select format from dropdown:
  - PDF - Best for sharing/printing
  - Markdown - Best for technical docs
  - HTML - Best for web
  - DOCX - Best for Word editing

**Configure**
- Select template (optional)
- Adjust quality settings
- Set page options

**Download**
- Click **Download**
- File saves to your computer

## Exporting a Project

**Open the project**
- Go to **Projects** page (`nav-projects`)
- Click on your project

**Compile and export**
- Click **Compile & Export**
- Select output format

**Configure compilation**
- Include table of contents: Yes/No
- Add chapter headers: Yes/No
- Page breaks between chapters: Yes/No
- Apply template: Select template

**Download**
- Click **Download**
- Compiled document saves to your computer

## Format Comparison

| Format | Pros | Cons |
|--------|------|------|
| PDF | Universal, printable, professional | Not easily editable |
| Markdown | Lightweight, version-controllable | Needs renderer for formatting |
| HTML | Web-ready, interactive | Requires hosting for sharing |
| DOCX | Fully editable in Word | May lose some formatting |

## Quality Settings

**High Quality**
- Full resolution images
- Crisp text rendering
- Larger file size
- Best for: Final documentation

**Medium Quality**
- Balanced image compression
- Good text clarity
- Moderate file size
- Best for: Review drafts

**Low Quality**
- Compressed images
- Smaller file size
- Best for: Quick previews

## Tips

**Before exporting**
- Review all content
- Check screenshots
- Verify step accuracy

**Choosing format**
- PDF for clients/external sharing
- Markdown for dev teams
- HTML for internal wikis

**Templates**
- Create branded templates for consistency
- Preview before final export
- Save common configurations
